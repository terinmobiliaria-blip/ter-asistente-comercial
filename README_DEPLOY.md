# TER Argumentario Comercial

Web estática lista para publicar en Vercel.

## Opción rápida

1. Entra a https://vercel.com.
2. Crea un proyecto nuevo.
3. Sube esta carpeta como proyecto estático.
4. Vercel generará un enlace público.

## Opción con GitHub

1. Sube la carpeta `ter-asistente` a un repositorio.
2. En Vercel, importa el repositorio.
3. Si el repositorio tiene más carpetas, selecciona `ter-asistente` como root directory.
4. No requiere comando de build.
5. No requiere carpeta de salida.

## Archivos necesarios

- `index.html`
- `styles.css`
- `app.js`
- `app-data.js`
- `vercel.json`
